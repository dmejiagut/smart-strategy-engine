Carpeta para imágenes/arte de VestPlan (logo, íconos, etc.).

LOGO OFICIAL
------------
Guarda aquí el logo con este nombre EXACTO:

    vestplan_logo.png

Recomendaciones:
- De preferencia con FONDO TRANSPARENTE (PNG), para que se vea bien sobre
  fondos claros y oscuros. Si tiene fondo blanco también funciona.
- Tamaño sugerido: 512x512 px o mayor (cuadrado).

Cuando lo pongas aquí, avísame y lo conecto en la pantalla de inicio de sesión
(y lo puedo usar también como ícono/favicon de la app).
